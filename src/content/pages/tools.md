<p>I build and maintain tools that support real-world translation workflows. Everything here is either open-source or source-available and built with working translators in mind.</p>

<h2>Supervertaler</h2>
<p><a href="https://supervertaler.com">Supervertaler</a> is an AI-enhanced CAT tool ecosystem with two products:</p>
<ul>
    <li><strong><a href="https://supervertaler.com/workbench/">Supervertaler Workbench</a></strong>&nbsp;– a free, open-source, cross-platform CAT tool (Windows, Mac, Linux) with multi-LLM translation, translation memory, glossary management, quality benchmarking, concordance search, voice commands, and integration with leading CAT tools. Includes two system-wide utilities: <strong>QuickMenu</strong> (right-click AI prompt launcher) and <strong>QuickTrans</strong> (instant multi-engine translation popup, similar to <a href="https://gt4t.ai/default/" target="_blank">GT4T</a>).</li>
    <li><strong><a href="https://supervertaler.com/trados/">Supervertaler for Trados</a></strong> – a Trados Studio plugin bringing inline contextual terminology display (TermLens) and AI-powered translation into the Trados ecosystem. Two paid tiers: TermLens (€10/month) for terminology features, and TermLens + Supervertaler Assistant (€15/month) adding AI chat, batch translate, and prompt library. 14-day free trial. <a href="https://github.com/Supervertaler" target="_blank">Source code</a> publicly available for security audit.</li>
</ul>

<h2>Beijerterm.com</h2>
<p><a href="https://beijerterm.com/" target="_blank">Beijerterm</a> is an open multilingual terminology database – one of the largest collections of specialist Dutch/English terminology available, built up over 30 years of professional translation work. Aimed at fellow translators and terminologists.</p>

<h2>WordCounter</h2>
<p><a href="https://github.com/michaelbeijer/WordCounter" target="_blank">WordCounter</a> is a free, open-source word counting engine for translators – built to challenge expensive CAT-era counting utilities with practical, transparent stats.</p>

<h2>Beijer.ink</h2>
<p><a href="https://beijer.ink/" target="_blank">Beijer.ink</a> is a personal note-taking web app with inline markdown styling, nested notebooks, full-text search, and a clean, responsive UI. Built as a single-user app, accessible as a PWA on mobile. <a href="https://github.com/michaelbeijer/Beijer.ink" target="_blank">Source on GitHub</a>.</p>
